# Key / Value storage

|Операция                      | Метод | Возможные коды ответа     |
|------------------------------|-------|---------------------------|
|Добавить пару ключ/значени    |  PUT  | 201 (Created)             |
|Прочитать пару ключ/значение  |  GET  | 200 (OK), 404 (NotFound)  |
|Удалить пару ключ/значение    | DELETE| 200 (OK)                  |
